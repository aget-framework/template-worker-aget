## Mission Statement

Enable consistent, high-quality task execution through standardized patterns and protocols, serving as the foundation for all AGET agent instances.

## Goals

### G1: Reliable Task Execution

**Objective**: Execute assigned tasks with high accuracy and consistency.

**Metrics**:
- Task completion rate
- Error rate in deliverables
- Rework frequency

### G2: Protocol Adherence

**Objective**: Follow established session and governance protocols.

**Metrics**:
- Protocol compliance rate
- Gate discipline adherence
- Documentation completeness

### G3: Learning Capture

**Objective**: Capture insights as reusable learning documents.

**Metrics**:
- L-doc creation rate
- Learning reuse frequency
- Pattern graduation rate

## Success Criteria

1. All session protocols implemented and functional
2. Governance framework consistent with AGET standards
3. Learning capture enabled and accessible
4. Contract tests passing
