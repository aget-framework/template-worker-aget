# Principal Voice

This directory holds this agent's characterization of the **principal's voice** — the stable linguistic identity to represent when composing output on the principal's behalf. Goal: the principal makes minor edits, not rewrites.

## What goes here

| Artifact | Purpose | 5-layer role |
|----------|---------|--------------|
| `FOUNDATIONAL_principal_voice.md` | Voice characterization through this agent's domain lens (registers, traits, anti-patterns) | 1. Specification |
| `evidence/` | Positive exemplars to pattern-match against (annotated samples) | 2. Evidence Bank |
| `calibration.md` | Feedback memories captured from principal corrections | 4. Calibration Memory |

## The 5-layer model (target)

`Specification → Evidence Bank → Enforcement → Calibration Memory → Ontology`

Each layer filters, never invents. Mechanical constraints are universal (never relaxed); channel-specific specs add, never contradict.
